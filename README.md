# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Application overview

A specialized internal web application designed for company operations, enabling automated data processing and integration with external services. The application serves as bridge between varios data sources and company workflows.

## Core Functionality 

### Data Processing 

Excel file manipulation using XLSX library for parsing and generating spreadsheets, structured data export capabilities for reporting and analysis.

### External Integrations

The application work with Bitrix24 integration for automated creation and management of smart processes, and **Axios**-based HTTP client for reliable API communication. 

### User Interface

made with modern React 19 and Material-UI v7 for professional UI components. It also use emotion for CSS-in-JS styling. **SweetAlert2** for professional user notifications and confirmations.

## Deployment

Configured for GitHub Pages deployment, making it accessible as an internal company tool with easy updates and maintenance.
